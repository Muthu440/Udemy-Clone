# 🎓 Udemy Clone Website

A responsive Udemy-inspired website built using HTML5 and CSS3. This project was created to practice frontend web development skills and understand modern website layouts.

## 🚀 Features

- Responsive Design
- Navigation Bar
- Course Categories Section
- Featured Courses
- Recommended Courses
- Modern UI Layout
- Footer Section
- Mobile-Friendly Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- Responsive Web Design
